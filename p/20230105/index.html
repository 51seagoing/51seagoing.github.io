快速上手WordPress：2小时打造专业网站全攻略

零基础也能轻松搭建，WordPress建站秘籍大公开！

Jan 05, 2023
阅读时长: 1 分钟

上手WP的痛点在于

过于丰富的插件生态和超级多的概念

使他变得臃肿和复杂学习曲线越来越陡峭

快速入门wordpress 必须记住下面这句话：找对插件，作对配置

所以本文重点介绍在每个特定的场景应该用什么插件，应该如何配置（大量截图）！建议收藏！

WP基础概念1分钟

需要重点介绍的是theme，page，shortcode：

theme 主题控制了一个页面的布局 和 视觉的框架（具体视觉是可以调整的）。

page 对比 post ，是可以动静结合的一个页面，可以穿插所有插件（通过组件 或者 shortcode 整合进页面）。一般我们发布文章都是发布post。

shortcode 可以被任意的页面任意的插件整合，非常的灵活。大部分好插件都支持生成 shortcode。

block 为一个页面上的一个局部（包含多个组件widget）

进一步进入正题（点题 标题的2小时是以下这些所有环节）：

域名 服务器

  • 域名：nameliso，namecheap，spaceship(首年价格便宜)
  • 服务器：hostinger 不完美但初期够用
    • 方便/自动备份/弹性升级/免费邮箱/CDN加速-结合wp插件
    • 如果资源很多 谨慎选择hostinger-web hosting套餐 他们的硬盘300G 无法升级！
    • 他的DNS不太行，建议换成其他的如CF 国内访问会块不少（后台编辑体验更高）。

如果偷懒直接用hostinger的ai服务 其实5分钟就可以生成一个站点，但是为了后期维护方便，使用最具性价比的插件还是建议按照本教程进行安装推荐的插件。

kadance 是推荐新手用的主体。他有ai建站的流程 点点点一路点就能生成专业美观的插件。

多语言的重要性是 能让网站流量快速的在几周内翻3-5倍：

多语言 用TranslatePress。

速度是用户体验的关键，也是Google 排名算法中，很关键的一点。无代码可以优化到100%。

优化需要的插件和配置：

  • 站点最后删除所有非必要插件
  • Imagify or Converter for Media 图片大小优化 Webp
  • Litespeed- Cache Cache& CDN打开（和hostinger搭配）
  • 如果服务器不是hostinger，可以用WP Rocket
  • Elementor-Setting-有Peformance标签选项打开
  • Index WP MySQL For Speed 启动high-performance keys
